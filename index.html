    Welcome to Magnolia DAO

    Welcome to Magnolia DAO documentation.

    What is Magnolia?

    Magnolia is a blockchain written from scratch using PHP, Python3 and MongoDB, designed so that all miners have an equal chance to mine a block regardless of his hashrate. This favors decentralization and eliminates the possibility of 51% attacks.

    It is also compatible with the main features of Ethereum, as well as the Metamask wallet. New features will be added in the future, to highlight:

    • Simple contracts without programming: The code for these contracts will be integrated into the server source code.
    • Contract Editor / Manager: Written in HTML5 and javascript, accessible from any browser with the Metamask extension.
    • Advanced contracts: Written in PHP, similar to EVM-compatible blockchain Solidity contracts.
    • Support for tokens, NFT and decentralized applications.
    • Staking: Stake your NOLIA for get governance tokens.

    Who is behind this project?

    Magnolia has a DAO structure, so it is the community that makes the decisions and chooses the priorities in the Roadmap. For this purpose, the NOLIA token can be staked to obtain governance tokens, which can be used to vote on the different proposals.

    The project is developed by five people linked to the crypto community who make up the core development team, to which new developers are always welcome.

    For the moment, the five of us prefer to keep our anonymity so that this does not interfere with our jobs and/or main projects.

    NOLIA was premined?

    NOPE! The development team only mined the genesis block on March 1, 2022 and 3 more blocks for test the network, for the value of 40 NOLIA.

    After that, no more block will be mined by team in first 24 hours. After that, we put to work a team with modest hardware (100 h/s) so that in case there are no more miners, blocks will continue to be generated.

    You can check this by visiting our address in the block explorer: https://magnolia.eslime.net/explorer/address.html?address=0x0fa10b766aD2F40110BE9ed1fEFdbcD2bDdB432F
